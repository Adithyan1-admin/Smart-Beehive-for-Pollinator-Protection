# Smart-Beehive-for-Pollinator-Protection
Smart Beehive for Pollinator Protection is an AI-powered beehive monitoring system that uses STM32, NanoEdge AI, ESP32, DHT22, and MQ135 sensors to detect bee distress, monitor hive conditions, and automate UV treatment and ventilation. Real-time alerts help beekeepers protect colonies, improve hive health, and support sustainable pollination.
