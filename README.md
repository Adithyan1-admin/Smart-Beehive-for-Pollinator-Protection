# Smart-Beehive-for-Pollinator-Protection
Smart Beehive for Pollinator Protection is an AI-powered beehive monitoring system that uses STM32, NanoEdge AI, ESP32, DHT22, and MQ135 sensors to detect bee distress, monitor hive conditions, and automate UV treatment and ventilation. Real-time alerts help beekeepers protect colonies, improve hive health, and support sustainable pollination.

The FET-Enabled Smart Beehive for Pollinator Protection is an AI-powered smart beehive monitoring system designed to improve bee health and prevent colony loss. The system uses STM32F411RE, NanoEdge AI, ESP32, and environmental sensors to continuously monitor bee activity and hive conditions. Bee buzzing sounds are analyzed in real time to classify hive behavior as Normal or Distress. When distress is detected continuously, the system automatically activates UV treatment and ventilation while sending alerts to the beekeeper through Telegram. This solution reduces manual inspections, enables early intervention, and supports sustainable beekeeping practices.
